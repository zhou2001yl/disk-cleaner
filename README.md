# 🧹 一键磁盘清理工具

一个 Windows 平台下的磁盘清理 GUI 工具，支持**垃圾文件清理**和**大文件查找**两大功能。

---

## 功能特性

### 1. 垃圾清理
扫描并清理以下类别的垃圾文件：

| 类别 | 说明 |
|------|------|
| Windows 临时文件 | %TEMP% 目录下的临时文件 |
| 回收站 | 各磁盘 $Recycle.Bin |
| 浏览器缓存 | Edge / Chrome 浏览器缓存 |
| 日志文件 (.log) | 系统及应用程序日志 |
| 缩略图缓存 | Windows 缩略图缓存 |
| Recent 文档历史 | 最近使用的文件列表 |
| Windows 更新缓存 | SoftwareDistribution 下载缓存（默认关闭） |
| 预读文件 (Prefetch) | 系统预读文件（默认关闭） |

- 可自由勾选需要清理的项目
- 扫描过程实时显示进度和已发现文件大小
- 清理前显示确认对话框，列出文件数量和可释放空间

### 2. 大文件查找
扫描指定磁盘中超过设定大小的文件：

- **可配置阈值**：默认 1GB，支持 100MB、500MB、1GB、2GB、5GB、10GB 或自定义
- **排除系统文件夹**：默认排除 Windows、Program Files 等系统目录
- **实时显示结果**：扫描过程中表格实时更新
- **文件操作**：
  - 📂 **打开位置** — 在资源管理器中定位文件
  - 🗑 **删除选中** — 永久删除选中的大文件
  - 📋 **复制路径** — 复制文件完整路径到剪贴板
  - 双击文件也可直接打开位置

## 使用方法

### 运行方式
```bash
python 磁盘清理工具.py
```
或直接双击 `磁盘清理工具.py` 运行。

### 建议
- **右键 → 以管理员身份运行**，否则部分系统目录无法访问
- 使用 Python 3.6+，无需安装第三方依赖（仅使用内置 tkinter、os、threading 等标准库）

---

## 界面截图

程序使用标签页布局：
- **「垃圾清理」标签页**：选择磁盘 → 勾选项目 → 扫描 → 清理
- **「大文件查找」标签页**：选择磁盘 → 设置大小阈值 → 扫描 → 操作文件

- GUI 框架：tkinter（Python 内置）
